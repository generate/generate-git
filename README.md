# generate-git [![NPM version](https://badge.fury.io/js/generate-git.svg)](http://badge.fury.io/js/generate-git)

> Generate a vanilla node.js project.

## Install

Install with [npm](https://www.npmjs.com/)

```sh
$ npm i generate-git --save
```

## Usage

